## 📜 Overview

Welcome to my personal portfolio repository! This is the source code for my hacker-themed portfolio and cybersecurity learning hub. Driven by a passion for exploring the depths of technology, this site showcases my journey from configuring Arch + Hyprland from scratch to diving deep into ethical hacking and penetration testing.

## 🚀 Features

- **Hacker-Themed UI:** Matrix rain background, glitch effects, scanlines, and hex overlays.
- **Interactive Terminal:** Custom built-in MASU terminal simulator for a fully immersive hacker experience.
- **Dynamic Content:** Showcases my toolkit, skills, and the active projects I'm building.
- **Cyber Academy Portal:** Access to curated free courses on networking, web hacking, privilege escalation, and more.
- **Fully Responsive:** Adapts seamlessly across mobile and desktop devices.

## 💻 Local Setup

Want to run this locally and poke around the terminal?

1. Clone the repository:
   ```bash
   git clone https://github.com/Maty156/portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in any modern browser, or use a live server:
   ```bash
   # If you have python3 installed:
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
